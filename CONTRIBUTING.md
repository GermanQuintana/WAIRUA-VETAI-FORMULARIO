# Contributing

Gracias por contribuir a la guia terapeutica.

## Flujo de contribucion

1. Crea una rama desde `main`.
2. Anade o edita una ficha terapeutica.
3. Incluye al menos una referencia cientifica verificable.
4. Abre un Pull Request con resumen clinico.

## Requisitos minimos para nueva entrada

- Principio activo.
- Al menos un nombre comercial.
- Especie(s) aplicable(s).
- Indicacion y dosis.
- Contraindicaciones principales.
- Nivel de evidencia.
- Referencias (DOI o URL cuando exista).

## Calidad cientifica

- Prioriza guias clinicas, revisiones sistematicas y estudios controlados.
- Si la evidencia es baja, indicalo de forma explicita en el campo `evidenceLevel`.
- Evita recomendaciones fuera de ficha tecnica sin justificacion.
